

        Confidential  >>  Secret  >>  Top Secret  


Confidential (ระดับลับต่ำสุด):
ข้อมูลที่อยู่ในระดับนี้จะถูกป้องกันจากการเปิดเผยต่อบุคคลภายนอกที่ไม่ได้รับอนุญาต หากข้อมูลนี้รั่วไหล อาจทำให้เกิดผลกระทบที่ไม่รุนแรงต่อความมั่นคงของประเทศ


Secret (ระดับลับ):
ข้อมูลที่มีความสำคัญต่อความมั่นคงของประเทศ หากข้อมูลนี้รั่วไหล อาจทำให้เกิดผลกระทบที่รุนแรงต่อความมั่นคงของประเทศ


Top Secret (ระดับสูงสุด):
ข้อมูลที่มีความสำคัญสูงสุดต่อความมั่นคงของประเทศ การรั่วไหลของข้อมูลระดับนี้อาจมีผลกระทบร้ายแรงหรือทำให้ประเทศตกอยู่ในอันตราย
